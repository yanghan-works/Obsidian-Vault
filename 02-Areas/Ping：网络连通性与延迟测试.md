## 核心概念

- 用来检查网络的第一步骤
- 像是在山谷喊了一声：“喂”
- 有人回复：“我在”，说明连接成功
- 如果没人回复，连接超时失败
- 公司家庭网络不通畅使用

## 用法参数

- [[ping -s 指定发送大小]]
- [[ping -c 指定发送次数]]

### 检查主机和网络设备的连通性

```shell
ping ip地址
```

### 检查主机和网卡的连通性

```shell
ping 127.0.0.1
```

### 检查主机和网关的连通性

```shell
ping 路由器IP
```

### 检查主机和外部网络的连通性

```shell
ping www.baidu.com
```
